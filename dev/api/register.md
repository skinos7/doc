

## 组件寄存器API说明

组件寄存器用于存放组件存放一些数据, 通过这些寄存器组件间可实时操作及交互的信息
此文档描述的函数接口都位于[land/register.h](./include/land/register.h)中, 通过包含[land/skin.h](./include/land/skin.h)即可
```c
    #include "land/skin.h"
```
